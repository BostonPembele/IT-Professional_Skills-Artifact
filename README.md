🛒 Grocify – Grocery Delivery Website


    📌 Project Overview


Grocify is a responsive grocery delivery website that allows users to browse products, add items to a cart, and complete a checkout process. The goal of this project is to demonstrate front-end development skills using modern tools and techniques.


    🚀 Features


🏠 Homepage with hero section and call-to-action buttons

📦 Product page with “Add to Cart” functionality

🛒 Shopping cart system using localStorage

💳 Checkout page with order summary and form validation

📱 Responsive design using Tailwind CSS


    🛠️ Technologies Used


HTML5

Tailwind CSS

JavaScript (Vanilla JS)

Local Storage (for cart persistence)


    Setup Instructions
    

Download or clone the project repository

Open the project folder in Visual Studio Code

Run the Tailwind CSS build command:

npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
Open home.html in your browser to start the project
